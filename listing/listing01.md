Что выведет программа? Объяснить вывод программы.

```go
package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}
```

Ответ:
```
Будет выведено [77, 78, 79], т.к. мы берём слайс от элемента с индексом 1 включительно до элемента с индексом 4 не
 включительно.

```
