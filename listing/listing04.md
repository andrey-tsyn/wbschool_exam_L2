Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Программа выведет последовательно числа от 0 до 9, после чего будет deadlock, т.к. мы вызываем в отдельной горутине цикл,
который записывает в небуферизированный канал числа и завершает работу, а в основной горутине постоянно считываются 
данные до тех пор, пока не закроется канал. Т.к. канал нигде не закрывается и все горутины не выполняют работу, то
возникает deadlock.
```
